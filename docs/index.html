mb-article HTML Format for Quarto

A journalistic format for Quarto posts and articles

mb-article is a Quarto extension for HTML formats with an opinionated journalistic design and simplified navigational elements, optimized for textual content on smaller display sizes. This custom format makes use of standard Quarto Project and HTML Article variables, such as branding, navigation links, and page banners, but changes the default layout and behaviors.

Typography

A rounded image moved to the margin.

1 About vertical rhythm and why it matters for content readability, see Klaas Leussink article on his personal blog.

Bootstrap default line-height and font sizing typically work, but this format enforces a strict vertical rhythm whenever possible1. I made sure to define all vertical spacing and padding as multiples of a standard line height of $spacer.

No additional changes to font definitions aside from using Crimson Pro and M+ Code fonts2.

2 M+ font system was designed by Japanese artist Coji Morishita with extended support for Latin, Japanese, Cyrillic. It is an excellent monospace font for smaller display sizes.

Here are bold, italic, and smaller text spans. Here is superscript and subscript text, and strikethrough notes. Semantic colors with e.g. .text-info provide accented text. I have added 2 background classes for my own use, a semi-transparent and blurred .bg-shaded and a .bg-striped. This theme uses .text-muted to de-emphasize content such as post metadata, margin notes, and captioning.
